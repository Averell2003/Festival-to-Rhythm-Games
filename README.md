# Festival-to-Rhythm-Games
Welcome to the Fortnite Festival to Rhythm Games Repository. Here you will find every single songs from Fortnite Festival for different game formats: Guitar Hero 2, Guitar Hero Warriors of Rock, Rock Band 1, 2 and Lego and finally ROCK BAND 3.

# MISSING SONG? 
Don't make any issues about reporting missing songs. This Repo will be updated from time to time.
So be patient.


# What game formats can be found for the songs?
The song format will be for GH2, GHWoR, RB1, 2, Lego and RB3.

# GH2 and WoR Warning
For the Festival to GH2 songs, don't forget to pack your songs before playing. because too many unpacked songs may corrupt your GH2 save file.
